# Content Studio - Advanced Marketing Content Generator

## What It Is

A professional content editor with real-time AI critique built into Content Predator. Generate and edit marketing content in multiple formats with instant feedback from Claude.

## Access

Dashboard → **CONTENT STUDIO** button
Or: http://localhost:3000/studio

## Features

### 1. Multiple Content Formats

**Emails:**
- Launch Email (Direct Response style)
- Feature Announcement
- Ben Settle Style Email (story-driven, contrarian)
- Gary Halbert Sales Letter (long-form, mechanism-heavy)

**Twitter:**
- Long Tweet (280-600 characters)
- Twitter Thread 10X (10-15 tweets, high value)
- Standard Tweet (250 chars)
- Standard Thread (6-8 tweets)

**Long-Form:**
- Landing Page (conversion-optimized)
- Long-Form Sales Page (2000+ words)
- Substack Article (outline + structure)

### 2. Split-Pane Editor

**Left Pane - Editor:**
- Live word/character count
- Copy to clipboard
- Clear content
- Manual editing with formatting

**Right Pane - Critique:**
- Real-time AI feedback
- What works / What fails
- Immediate fixes with exact replacements
- Conversion score (1-10)
- Brutal one-line verdict

### 3. Ben Settle / Gary Halbert Styles

The marketing-writer skill is fully integrated:

**Ben Settle Emails:**
- Contrarian takes
- Story-driven
- Email cliffhangers
- Entertaining first, selling second
- Teases tomorrow's email

**Gary Halbert Letters:**
- "Reason why" copy
- Mechanism explanation
- Bullet-heavy benefits
- Social proof with numbers
- Strong guarantees

### 4. Real-Time Critique

Click **CRITIQUE** to get:
- Specific line-by-line feedback
- Conversion-killing problems identified
- Exact text replacements
- Jargon/BS called out
- Conversion score
- One-line brutal verdict

## Usage Workflow

### Quick Generate

1. Dashboard shows opportunities from scans
2. Click **CONTENT STUDIO**
3. Select opportunity from dropdown
4. Choose format (email, tweet, landing page, etc.)
5. Click **GENERATE**
6. Wait 10-30 seconds
7. Content appears in editor

### Edit & Critique Loop

1. Edit generated content manually
2. Click **CRITIQUE** for feedback
3. See specific problems + fixes
4. Apply suggested changes
5. Critique again
6. Repeat until score is 8+/10

### Copy & Use

1. Content perfected?
2. Click **COPY**
3. Paste into email/X/Substack
4. Post

## Critique Scoring

**9-10/10:** Ship it. Converts.
**7-8/10:** Close. Minor fixes needed.
**5-6/10:** Needs work. Major revisions.
**1-4/10:** Start over. Fundamentally broken.

## Copy Principles (Built-In)

All generated content follows:
- Lead with pain, not product
- Specific numbers over vague claims
- "You" 3x more than "we"
- One clear CTA
- No corporate speak
- Grade 8 reading level
- Short punchy sentences
- Confrontational tone for Biblical Man brand

## Keyboard Tips

- **Tab** in editor = indent
- **Cmd/Ctrl + A** = select all
- **Cmd/Ctrl + C** = copy (or use COPY button)
- **Cmd/Ctrl + Z** = undo edits

## Tips for Best Results

### Before Generating
- Pick the right opportunity (8-10 score)
- Choose format that matches goal
- Know your specific offer/CTA

### While Editing
- Keep sentences under 20 words
- Remove every adverb
- Replace jargon with plain language
- Add specific numbers/metrics

### When Critiquing
- Don't argue with feedback
- Apply fixes exactly as suggested
- Critique again after major edits
- Aim for 8+/10 before shipping

## Troubleshooting

**"No opportunities" in dropdown:**
→ Run a scan first from Dashboard

**Generation taking forever:**
→ Normal for sales_page (30-45 seconds)
→ Other formats: 10-30 seconds

**Critique seems generic:**
→ Content might be too short
→ Try with 200+ words minimum

**Can't copy to clipboard:**
→ Browser permission issue
→ Manually select all and copy
